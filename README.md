# VoidView

VoidView는 광고-focused 웹사이트 프로젝트입니다. 이 프로젝트는 사용자 경험을 개선하고 광고 관련 정보와 데이터를 효율적으로 제공하는 것을 목표로 합니다.

## 기능

- 광고 콘텐츠 표시
- 사용자 맞춤형 광고 추천
- 광고 통계 대시보드
